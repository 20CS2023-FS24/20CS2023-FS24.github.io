Sets#

A set in Python is an unordered collection of unique elements. Sets are mutable, which means you can add or remove items after the set has been created. Sets are used to perform mathematical set operations like union, intersection, symmetric difference, etc. Sets are used to eliminate duplicate entries from a list or a tuple. Sets are also used to perform common mathematical operations like union, intersection, difference, and symmetric difference.

Unlike lists or tuples, sets do not have an order. When you print a set, the order of the elements will be different than the order in which you added them.

Note

There is an alternative to sets called frozenset which is immutable.

Set Creation#

You can create a set by placing a comma-separated sequence of elements within curly braces {}. Alternatively, you can use the set() constructor.

# Creating a set using curly braces
my_set = {1, 2, 3, 4, 5}
print("Set created with curly braces:", my_set)

# Creating a set using the set() constructor
another_set = set([1, 2, 3, 4, 5])
print("Set created with set() constructor:", another_set)

Set created with curly braces: {1, 2, 3, 4, 5}
Set created with set() constructor: {1, 2, 3, 4, 5}

Modifying Sets#

Sets are mutable, so you can add or remove elements after the set is created.

# Adding an element
my_set.add(6)
print("After adding 6:", my_set)

# Removing an element
my_set.remove(2)
print("After removing 2:", my_set)

# Discarding an element (doesn't raise an error if the element is not found)
my_set.discard(10)  # 10 is not in the set, but this won't raise an error
print("After discarding 10 (which wasn't in the set):", my_set)

# Popping an arbitrary element
popped = my_set.pop()
print(f"Popped element: {popped}")
print("After popping:", my_set)

# Clearing the set
my_set.clear()
print("After clearing:", my_set)

After adding 6: {1, 2, 3, 4, 5, 6}
After removing 2: {1, 3, 4, 5, 6}
After discarding 10 (which wasn't in the set): {1, 3, 4, 5, 6}
Popped element: 1
After popping: {3, 4, 5, 6}
After clearing: set()

Set Methods and Operations#

Sets support a variety of mathematical operations such as union, intersection, difference, and symmetric difference.

set1 = {1, 2, 3, 4}
set2 = {3, 4, 5, 6}

# Union
union_set = set1.union(set2)
print("Union:", union_set)

# Intersection
intersection_set = set1.intersection(set2)
print("Intersection:", intersection_set)

# Difference
difference_set = set1.difference(set2)
print("Difference (set1 - set2):", difference_set)

# Symmetric difference
sym_diff_set = set1.symmetric_difference(set2)
print("Symmetric difference:", sym_diff_set)

Union: {1, 2, 3, 4, 5, 6}
Intersection: {3, 4}
Difference (set1 - set2): {1, 2}
Symmetric difference: {1, 2, 5, 6}

Set Relationships#

You can check if a set is a subset or superset of another set using the issubset() and issuperset() methods.

set_a = {1, 2, 3, 4, 5}
set_b = {1, 2, 3}
set_c = {1, 2, 3, 4, 5, 6}

# Subset
print("Is set_b a subset of set_a?", set_b.issubset(set_a))

# Superset
print("Is set_a a superset of set_b?", set_a.issuperset(set_b))

# Equality
print("Are set_a and set_c equal?", set_a == set_c)

Is set_b a subset of set_a? True
Is set_a a superset of set_b? True
Are set_a and set_c equal? False

Iterating Over Sets#

You can iterate through the elements of a set using a for loop. Remember that sets are unordered, so the iteration order is not guaranteed.

my_set = {1, 3, 5, 7, 9}

print("Iterating through the set:")
for item in my_set:
    print(item, end=" ")
print()  # New line

Iterating through the set:
1 3 5 7 9

Frozen Sets#

A frozenset is an immutable version of a set. Once created, you cannot add or remove elements from a frozenset. Frozen sets are useful when you need to use a set as a key in a dictionary or as an element in another set (you cannot nest sets, but you can nest a frozenset).

To create a frozenset, use the frozenset() constructor.

# Creating a frozenset
frozen_set = frozenset([1, 2, 3, 4, 5])
print("Frozen set:", frozen_set)

# Attempting to modify a frozen set (this will raise an error)
try:
    frozen_set.add(6)
except AttributeError as e:
    print("Error:", e)

Frozen set: frozenset({1, 2, 3, 4, 5})
Error: 'frozenset' object has no attribute 'add'

Matplotlib#

What is Matplotlib? {.hide .smaller-85}#

Matplotlib is a comprehensive library for creating static, animated, and interactive visualizations in Python. It is widely used for scientific computing and data visualization. Key features include:

  • High-quality 2D and 3D plots

  • Support for various output formats (e.g., PDF, PNG, SVG)

  • Fine-grained control over all aspects of presentation

References {.hide}#

The following resources from the official Matplotlib documentation provide additional information on the library:

  • Matplotlib Homepage

  • Matplotlib Documentation

Installation {.hide}#

You can install Matplotlib using pip:

pip install matplotlib

Or using conda:

conda install matplotlib

Rendering Environment {.hide}#

Matplotlib can render plots in various environments. Sometimes, it may assume a GUI is available when it is not. To avoid issues on systems without a display, set the backend to Agg. This non-interactive backend is ideal for generating plots as image files or for batch processing.

import matplotlib
matplotlib.use('Agg')  # Use 'Agg' for non-GUI environments

Plotting with Matplotlib {.hide}#

Matplotlib provides two interfaces for creating plots:

  • MATLAB-style interface: Convenient for quick and simple plots.

  • Object-oriented interface: Offers more control and flexibility over plot elements.

MATLAB-style Interface {.hide}#

The MATLAB-style interface is provided by the pyplot module. It is a collection of functions that make Matplotlib work like MATLAB.

The standard way to import the pyplot module is as follows:

import matplotlib.pyplot as plt

Example: MATLAB-style Interface {.hide .smaller-70}#

This demonstrates the MATLAB-style interface.

import matplotlib.pyplot as plt
import numpy as np

x = np.linspace(0, 10, 200)
y = np.sin(x)

plt.plot(x, y, 'b-', linewidth=2)
plt.show()

Example: Object-oriented Interface {.hide .smaller-70}#

The object-oriented interface is more powerful and flexible than the MATLAB-style interface. It involves creating figure and axis objects explicitly and calling methods on them.

import matplotlib.pyplot as plt
import numpy as np

x = np.linspace(0, 10, 200)
y = np.sin(x)

fig, ax = plt.subplots()
ax.plot(x, y, 'b-', linewidth=2)
plt.show()

Customization: Ticks, Tick Labels, and Titles {.hide}#

You can customize the ticks, tick labels and titles of the plot using the set_xticks(), set_xticklabels(), set_xlabel(), and set_title() methods.

fig, ax = plt.subplots()
ax.plot(x, y, 'r-', linewidth=2, label='sine function', alpha=0.6)
ax.legend(loc='upper center')
ax.set_yticks([-1, 0, 1])
ax.set_yticklabels(['min', 'mid', 'max'])
ax.set_title('Test plot')
plt.show()

Multiple Plots on One Axes {.hide}#

This example demonstrates how to create multiple plots on one axes, by calling the plot() method multiple times on the same ax object.

It also shows the use of LaTeX in the plot labels.

import matplotlib.pyplot as plt
import numpy as np
from scipy.stats import norm
from random import uniform

fig, ax = plt.subplots()
x = np.linspace(-4, 4, 150)
for i in range(3):
    m, s = uniform(-1, 1), uniform(1, 2)
    y = norm.pdf(x, loc=m, scale=s)
    current_label = f'$\\mu = {m:.2}$'
    ax.plot(x, y, linewidth=2, alpha=0.6, label=current_label)
ax.legend()
plt.show()

Multiple Subplots in One Figure {.hide}#

This example demonstrates how to create multiple subplots in one figure using the subplots() function and specifying the number of rows and columns. The figsize argument sets the size of the figure (width, height).

import matplotlib.pyplot as plt
from scipy.stats import norm
from random import uniform

num_rows, num_cols = 3, 2
fig, axes = plt.subplots(num_rows, num_cols, figsize=(10, 12))
for i in range(num_rows):
    for j in range(num_cols):
        m, s = uniform(-1, 1), uniform(1, 2)
        x = norm.rvs(loc=m, scale=s, size=100)
        axes[i, j].hist(x, alpha=0.6, bins=20)
        t = f'$\\mu = {m:.2}, \\quad \\sigma = {s:.2}$'
        axes[i, j].set(title=t, xticks=[-4, 0, 4], yticks=[])
